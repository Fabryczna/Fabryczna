# Project Summary
Kawiarnia Fabryczna is a modern, industrial-themed cafe website designed to promote a specialty coffee shop located in the heart of Warsaw. The website provides essential information about the cafe's unique offerings, social mission, and community engagement, ensuring an engaging and user-friendly experience optimized for various devices.

# Project Module Description
The project consists of several functional modules:
- **Navigation**: Provides links to different sections of the website.
- **Hero**: Displays the cafe's name with an engaging background image.
- **About**: Shares the story and mission of the cafe.
- **Menu**: Lists the cafe's offerings, including specialty coffee, pastries, and more.
- **Gallery**: Showcases real images of the cafe and its products.
- **Contact**: Provides contact information, including address, phone number, and operating hours.

# File Description Inventory
- **README.md**: Overview of the project and setup instructions.
- **index.html**: Entry point of the application.
- **src/components/**: Contains all React components for the application.
- **src/i18n/**: Internationalization setup files for language support.
- **public/assets/**: Contains images and other static assets.
- **vite.config.ts**: Configuration file for Vite, the build tool.

# Technology Stack
- **React**: JavaScript library for building user interfaces.
- **TypeScript**: Superset of JavaScript for type safety.
- **Vite**: Build tool for fast development.
- **Shadcn-ui**: UI component library for modern interfaces.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **i18next**: Internationalization framework for multilingual support.

# Usage
1. Install dependencies:
   ```bash
   pnpm install
   ```
2. Build the project:
   ```bash
   pnpm run build
   ```
3. Run lint checks:
   ```bash
   pnpm run lint
   ```
